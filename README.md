# Peekachoo Backend

A simple Express.js backend API.

## Installation

```bash
npm install
```

## Running the Server

```bash
# Production
npm start

# Development (with auto-reload)
npm run dev
```

## Endpoints

- `GET /health` - Health check
- `GET /api` - API welcome message
- `GET /api/peekachoo` - Sample endpoint

## Environment Variables

- `PORT` - Server port (default: 3000)
